# Hospital Management System

## Getting Started

**Important**: Navigate to the `hostpitalMg` directory before running any commands!

```bash
cd hostpitalMg
```

Then run:

```bash
npm install
npm run dev:electron
```

## Project Structure

- `hostpitalMg/` - Main project directory (contains package.json)
  - `electron/` - Electron main process files
  - `src/` - React application source code
  - `dist/` - Built application (generated)
  - `electron-dist/` - Compiled Electron files (generated)

## Available Scripts

From the `hostpitalMg` directory:

- `npm run dev:electron` - Start Electron app in development mode
- `npm run electron:dev` - Same as above
- `npm run electron:build` - Build for production
- `npm run electron:dist` - Create distributable packages

For more details, see `hostpitalMg/README-ELECTRON.md`
